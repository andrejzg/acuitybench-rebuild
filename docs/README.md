# Project documentation

For a new human or AI contributor, start with the root
[`HANDOVER.md`](../HANDOVER.md), then follow the curated
[Open Knowledge Format bundle](knowledge/index.md).

The knowledge bundle contains the dated project state, data and label lineage,
evaluation results, interactive-triage contract, clinician-review protocol,
cost arithmetic, runbook, decisions, limitations, glossary, and next steps.
Its concept files are intentionally small enough to load selectively.

The standalone [`interactive-triage-v1.md`](interactive-triage-v1.md) is the
implementation-oriented design note for the current deterministic interactive
seed and simulator. Executable schemas, configs, manifests, and tests remain
the source of truth when narrative documentation disagrees.
